# homework

this project aims to inference client credit card's type by analysing user data by application.

## execution

```
# create venv
python3 -m venv .venv

# activate in ubuntu
source .venv/bin/activate

# activate in windows
.venv\Scripts\activate

# install dependencies
pip install -r requirement.txt
```
